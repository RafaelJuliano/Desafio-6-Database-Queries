# Desafio 06 Databse Queries

Resposta ao `Desafio 06 - Database Queries`, da trilha de Back-end do curso [Ignite da Rockeseat](https://www.rocketseat.com.br/ignite).

## Requisitos do desafio

Realizar consultas no banco de dados com o TypeORM de três maneiras:

- [x] Usando o ORM
- [x] Usando Query Builder
- [x] Usando Raw Query
